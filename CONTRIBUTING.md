# Contributing to Virtual Digital Logic Lab

Thank you for your interest in contributing! We welcome improvements to the experiments, tutor logic, and UI.

## How to Contribute

1.  **Fork the Repository**: Create your own copy of the project.
2.  **Create a Branch**: `git checkout -b feature/AmazingFeature`
3.  **Make Changes**:
    *   Add new experiments in `units/`.
    *   Enhance the tutor in `tutor.py`.
    *   Improve visualizations in `circuits.py`.
4.  **Test**: Ensure the application runs locally with `python -m streamlit run main.py`.
5.  **Commit**: `git commit -m 'Add some AmazingFeature'`
6.  **Push**: `git push origin feature/AmazingFeature`
7.  **Open a Pull Request**: Describe your changes and why they are valuable.

## Code Style

*   Follow PEP 8 guidelines for Python code.
*   Use descriptive variable names.
*   Add comments for complex logic, especially in `circuits.py`.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on GitHub.
